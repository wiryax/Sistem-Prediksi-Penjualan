Petunjuk Penggunaan Aplikasi

Baca petunjuk penggunaan aplikasi dibawah ini.

  • 1. Input data jumlah customer dan data jumlah penjualan yang ingin diprediksi. data harus terdiri dari angka !!!
  • 2. untuk menambah jumlah kolom input, silahkan tekan tombol dengan icon "+"
  • 3. untuk mengurangi jumlah kolom input, silahkan tekan tombol dengan icon "-"
  • 4. setelah melakukan input data pastikan data yang diinput memiliki jumlah yang sama
  • 5. selanjutnya tekan tombol submit untuk memproses data prediksi
  • 6. pada bagian Forecasting Calculation akan menampilkan table hasil prediksi yang akan terjadi pada periode yang akan datang
  • 7. anda juga bisa menginputkan angka prediksi jumlah customer pada periode yang akan datang pada input field yang akan muncul setelah tombol submit pada input data ditekan
  • 8. hasil dari inputan yang diisi secara manual dapat dilihat pada bagian bawah input field dengan keterangan "hasil"

Pertanyaan Terkait

  • Apakah saya bisa menginputkan huruf pada bagian input data ?
    • Anda tidak bisa menginputkan data dengan tipe huruf / karakter, hal ini dikarnakan metode pemprediksian hanya bisa mengkalkulasikan data dengan tipe numerik / angka. sistem aplikasi ini juga sudah dirancang agar inputan yang masuk harus bertipe numerik.
  • Muncul keterangan "Data Invalid !!!"
    • Ini terjadi karna jumlah data antar data Jumlah Customer dengan data Jumlah Penjualan yang anda inputkan tidak sama / jumlah dari salah satu data kurang.
